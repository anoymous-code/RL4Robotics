# 2026-07-18 (深夜) · 0/20 之后：模仿学习失败的三层解剖

## 事情经过

[上一篇](2026-07-18-act.md)训练的 ACT-lite 在评测中**全军覆没：0/20**。
训练曲线明明很漂亮（L1 0.34 → 0.019）。这篇记录完整的破案过程——
对初学者而言，这次失败比一次顺利的成功值钱得多。

## 第一层：策略没有得到任务信息（谁都救不了）

三层诊断（`diagnose_act.py`）第一轮发现：

- 训练分布内预测很准（在演示自己的观测上，误差 0.008~0.03 rad）→ 模型学到了演示；
- 但目标格（前排 3f / 后排 3b）**在图像里根本看不出来**——撕之前板是完整对称的！
  策略没有任何依据判断该撕哪格，只能输出两类演示的**平均动作**——手伸向两格中间。

**修复**：目标格行号作为可学习 embedding token 喂给 Transformer——
这就是 VLA 里"语言指令"的最简形式。同时修了两个训练-推理不一致：
推理观测过一遍 JPEG 编解码对齐训练分布；换 ACT 论文的时间集成推理。

重训后依然 **0/20**。

## 第二层：环境缺了物理规则（重放实验立功）

换一个更狠的对照实验（`replay_test.py`）：**在 Gym 环境里逐帧重放演示的动作序列**。
连"完美动作"都失败——问题根本不在策略，在**环境**！

排查发现 `pill_env.py` 的 `step()` 只做物理步进，而采集演示时脚本专家身上挂着
两条"环境规则"：左爪的抓取锁定（sticky gripper）和易撕线的断裂判定。
它们写在专家的编排代码里，没进环境——**专家自带了一部分"世界"，
环境里那部分世界是缺失的**。策略/重放在这个环境里永远撕不断板。

**修复**：把两条规则移进环境物理——

- 闭爪 + 两指触到手柄 → 自动锁定；张爪 → 解除；
- 易撕线断裂需要**右爪夹住目标格 + 载荷持续超阈值**（纯搬运冲击不撕裂，
  这比原来"瞬时超阈值即断"更贴近韧性材料，也顺手修掉了重放时
  提板瞬间就把整板撕碎的 bug）。

修复后**重放成功**（撕剪入盒 B ✓ 回槽 ✓）——环境与数据终于自洽。策略依然 0/20。

## 第三层：相位混淆（模仿学习的经典陷阱）

相位诊断（`diagnose_rollout.py`）：rollout 时每 50 步用 qpos 最近邻定位
"当前进行到演示的百分之几"。结果触目惊心：

```text
step   0   相位  1%    正常起步
step 100   相位  5%    qpos 偏差开始积累（复合误差）
step 200   相位 94%    ！策略认为任务快结束了
step 550+  相位 97%    死锁：输出结束姿态，纹丝不动到超时
```

两个互相咬合的原因：

1. **观测歧义**：episode 开头（双臂中立位）和结尾（双臂中立位）的图像几乎一样，
   qpos 完全一样——唯一区别是板上缺不缺一格，在头部相机里只有几个像素。
   轨迹一旦偏离，策略把"开始"认成"结束"，直接进入收尾姿态；
2. **qpos 被忽略**：诊断显示 qpos 加噪 σ=0.3 rad，输出只变 0.002——模型完全
   靠图像。因为演示里图像与 qpos **完全冗余**（确定性专家，看图就知道关节角），
   梯度选择了信息更丰富的图像，qpos 通路萎缩。而图像恰恰在首尾是歧义的。

这就是教科书里 behavior cloning 的**复合误差 + 因果混淆**，在我们自己的任务上
活生生长了出来。

**对策**（流水线 v2，运行中）：

- **DART 带噪专家**：采集时给执行注入 OU 噪声（σ=4 mrad），但录制的标签是
  干净目标——演示从"一条细线"变成"带纠错的管道"，策略学会往回拉；
  噪声还打破了图像-qpos 的完全冗余（执行位形 ≠ 命令目标）；
- **传感器 dropout**：训练时 20% 概率抹掉整组图像，逼模型建立 qpos 通路——
  qpos 单调推进，是天然的"相位时钟"；
- 带噪专家验证 3/3 成功，150 条新数据采集 → 重训 → 评测的接力流水线已启动。

## 学到了什么

- **训练损失低 ≠ 策略能用**：BC 的训练分布是"专家走过的状态"，rollout 的
  分布是"策略自己走出的状态"——两者的差距（distribution shift）才是主战场；
- **重放实验是环境正确性的试金石**：任何 Gym 化改造后，第一件事应该是
  "重放专家动作能不能成功"——我们把它补成了管线的固定回归测试；
- **专家逻辑与环境物理必须划清界线**：专家可以"聪明"，但世界规则
  （抓住会锁定、拉扯会撕裂）必须住在环境里，否则策略学到的世界是残缺的；
- **观测设计要保证可区分性**：首尾同像的任务要么给策略记忆（时序模型），
  要么给足线索（qpos 通路、任务进度 token）。
